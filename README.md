# NovaVet

Landing page em React + Vite para uma clínica veterinária.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Gerar versão para Hostinger

```bash
npm run build
```

Depois, envie o conteúdo da pasta `dist` para a pasta `public_html` do domínio ou subdomínio na Hostinger.

## Personalização rápida

- Conteúdo: `src/config/content.js`
- Serviços: `src/config/services.js`
- Equipe: `src/config/team.js`
- Estrutura/galeria: `src/config/portfolio.js`
- Processo: `src/config/process.js`
- Depoimentos: `src/config/testimonials.js`
- FAQ: `src/config/faq.js`
- Cores e tipografia: `src/config/theme.js`
- SEO: `src/config/seo.js`
- Seções ativas: `src/config/modules.js`
